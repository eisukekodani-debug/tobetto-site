# 当別当育苗 サイト（試作）

鳥取の育苗農家「当別当育苗」の **会社案内・採用・商品紹介** を兼ねた静的サイトの試作。
購入はBASEのオンラインショップ（`https://tbtseedling.base.shop`）へ送客する構成。

> ⚠️ これはイメージ確認用の試作です。写真・価格・所在地・募集要項は **準備中の仮内容** を含みます。
> 過度な演出は避け、シンプルで誠実なトーンを優先しています。

## 構成

| ファイル | 内容 |
|---|---|
| `index.html` | トップ（会社案内＋商品＝BASE導線＋採用導線） |
| `recruit.html` | 採用ページ（働き方＝Value、募集要項の枠） |
| `style.css` | 共通スタイル（外部依存なし・システムフォント） |

依存ライブラリなし。画像は当面インラインSVGの「写真 準備中」プレースホルダー。撮影後に差し替える。

## ローカル確認

```bash
cd site
python3 -m http.server 8000
# → http://localhost:8000
```

## GitHub Pages で公開する手順（後日）

1. GitHubで空のリポジトリを作成（例: `tobetto-web`）
2. この `site/` の中身をリポジトリ直下に push
3. リポジトリの Settings → Pages → Branch を `main` / `/(root)` に設定
4. 数分後 `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開
5. 独自ドメイン `tobetto.com` を使う場合は Pages のカスタムドメインに設定し、DNSのCNAME/Aレコードを向ける
